Aurelia Release Notes - Mid August 2017

Posted by AureliaEffect on August 14, 2017

Today, we have a small set of release to update you with. We're gearing up for some big releases next. Stay tuned for more...

Minor Releases

Minor updates include new features, updates to bugs and performance improvements.

aurelia-webpack-plugin 2.0.0-rc.3

Features

  • support Webpack 3 ModuleConcatenationPlugin
  • support resources inside @inlineView

Bug Fixes

  • recognize router-view attributes
  • better control over entry sequence
  • chunk not supported by ModuleDependenciesPlugin
  • no default entrypoint with DLLPlugin
  • use browser-pal for electron-renderer (#100)

aurelia-ux 0.3.0

Features

  • expose style engine ( f623c70 ), closes #73
  • add ux-list control (#82) ( ee64238 )
  • add unchecked value ( 88505d9 ), closes #69

Bug Fixes

  • disabled checkbox behavior (#80) ( 32eb94c )
  • bundle all files with aurelia-cli ( c69c1ff ), closes #17
  • add disable button behavior (#76) ( a16f47f )
  • remove quotes ( 66966e4 )

Patch Releases

Patch updates include only fixes to bugs and performance improvements.
