Aurelia at Build 2017

If you are attending Build and are interested in seeing how to combine ASP.NET Core and Aurelia , you'll want to attend "Getting Started with Aurelia and ASP.NET Core" this Friday.

In this talk you’ll get a crash course on using Microsoft’s tools to build a new Aurelia ASP.NET Core app. We’ll start by creating the project from scratch and then learn how to use Aurelia’s core concepts, all while watching our application update in realtime in the browser as we develop. The session is on Friday at 12pm and is located in Tech Talk B.

We're also having an Aurelia Meetup in Seattle on Tuesday evening. If you are interested in attending, please contact @mobilemancer on Twitter.

Hope to see you at one of these events!
