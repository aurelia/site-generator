getOrCreateOwn Method

getOrCreateOwn( metadataKey : string , Type : Function , target : Function , targetKey ?: string ) : Object

Gets metadata specified by a key on a target, or creates an instance of the specified metadata if not found.

Params

metadataKey : string
The key for the metadata to lookup or create.
Type : Function
The type of metadata to create if existing metadata is not found.
target : Function
The target to lookup or create the metadata on.
targetKey ?: string
The member on the target to lookup or create the metadata on.
