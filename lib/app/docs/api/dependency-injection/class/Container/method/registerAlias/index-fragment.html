registerAlias Method

Registers an additional key that serves as an alias to the original DI key.

registerAlias( originalKey : any , aliasKey : any ) : Resolver

Registers an additional key that serves as an alias to the original DI key.

Params

originalKey : any
The key that originally identified the dependency; usually a constructor function.
aliasKey : any
An alternate key which can also be used to resolve the same dependency as the original.

Returns

The resolver that was registered.
